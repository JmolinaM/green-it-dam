# P0 — ¿Cuánto contamina tu código? (Concienciar)

Práctica de apertura transversal del módulo. El alumnado usa el método **Jigsaw** para investigar el impacto ambiental del sector TIC y calcula su propia huella de datos móviles.

## Duración

2 sesiones × 50 min

## Materiales

- `tabla_equivalencias_co2.pdf` — tabla imprimible de equivalencias de CO₂ (a añadir)
- `ficha_b_jigsaw_grupal.pdf` — ficha de síntesis grupal (a añadir)
- `ficha_c_huella_personal.pdf` — ficha individual de huella digital (a añadir)

## Recursos web utilizados en el Jigsaw

- [Electricity Maps](https://app.electricitymaps.com) — intensidad de carbono de la red eléctrica
- [Website Carbon Calculator](https://www.websitecarbon.com) — huella de webs
- [ML CO₂ Impact](https://mlco2.impact.ml) — huella de modelos de IA
- [PNAV Calculadora](https://algoritmosverdes.gob.es/es/recursos/calculadora-pnav-0) — calculadora oficial MITMA

## No requiere código Python

Esta práctica no incluye código. Véase la memoria del TFM (§4.2) para el guion completo.
